hegi-docker-wildlfy
===================

Author: Bernd Hegmanns

Aufbau
------
Das Projekt ist ein Multiprojekt, bestehent aus einem einfachen RS-Projekt sowie einem Docker-Projekt.

Anleitung
---------

1. Öffne eine Shell Basisverzeichnis/Projektverzeichnis
2. Führe aus:

		mvn clean install
3. Rufe die folgende URL auf (im Browser, oder einer anderen Anwendung):

		http://localhost:8080/restful-service/api/add/3/5
